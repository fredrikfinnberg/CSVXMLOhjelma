# CSVXMLOhjelma
Virta Julkaisut CSV-XML muuntotyökalu

CSV-XML-muuntotyökalun avulla OKM:n tiedonkeruuvaatimusten mukaisen CSV-tiedoston voi helposti muuttaa VIRTA-julkaisuskeeman mukaiseksi XML-tiedostoksi. Työkalu on tarkoitettu erityisesti niille organisaatioille, joiden järjestelmistä ei automaattisesti saa muodostettua VIRTA-skeeman mukaista XML-tiedostoa.
